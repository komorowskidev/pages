# Polityka prywatności (PL)

## Polityka prywatności

Data ostatniej aktualizacji: 23 maja 2026

Aplikacja służy do organizowania zakupów i została zaprojektowana z poszanowaniem prywatności
użytkownika.

## Jakie dane zbieramy

Aplikacja:

* nie wymaga zakładania konta,
* nie wymaga logowania,
* nie zbiera danych osobowych,
* nie wysyła żadnych danych przez Internet,
* nie śledzi aktywności użytkownika,
* nie korzysta z narzędzi analitycznych ani reklamowych.

Wszystkie dane wprowadzane przez użytkownika są przechowywane wyłącznie lokalnie na urządzeniu
użytkownika przy użyciu lokalnej bazy danych Room.

## Dostęp do Internetu

Aplikacja nie komunikuje się z serwerami zewnętrznymi i nie przesyła żadnych danych poza urządzenie
użytkownika.

## Bezpieczeństwo danych

Dane zapisane w aplikacji pozostają wyłącznie na urządzeniu użytkownika. Usunięcie aplikacji może
spowodować usunięcie wszystkich zapisanych danych.

## Zmiany w polityce prywatności

Polityka prywatności może być aktualizowana w przyszłości w związku z rozwojem aplikacji lub
zmianami prawnymi. Aktualna wersja będzie zawsze dostępna pod tym samym adresem.

# Privacy Policy (EN)

## Privacy Policy

Last updated: May 23, 2026

This application is designed to help users organize shopping and was created with user privacy in
mind.

## Data Collection

The application:

* does not require account creation,
* does not require login,
* does not collect personal data,
* does not send any data over the Internet,
* does not track user activity,
* does not use analytics or advertising services.

All data entered by the user is stored exclusively locally on the user's device using the Room local
database.

## Internet Access

The application does not communicate with external servers and does not transmit any data outside
the user's device.

## Data Security

All application data remains stored locally on the user's device. Uninstalling the application may
result in deletion of all stored data.

## Changes to This Privacy Policy

This Privacy Policy may be updated in the future due to application development or legal changes.
The latest version will always be available at the same address.

## Contact

If you have any questions regarding privacy, you may contact the developer at:

`komorowskidev@gmail.com`
